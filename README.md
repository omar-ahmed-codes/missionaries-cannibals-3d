# ⛵ Missionaries & Cannibals — 3D

### [🌐 Play Live → missionaries-cannibals-3d.vercel.app](https://missionaries-cannibals-3d.vercel.app)

A visually stunning **3D puzzle game** built with **Three.js** — the classic AI river-crossing problem, reimagined with animated water, stylized characters, and smooth boat-crossing animations.

> **AI Semester IV Project**

## 🎮 The Puzzle

Get **3 missionaries** and **3 cannibals** across the river. The catch?

- 🚣 The boat carries **1–2 people** at a time
- ⚠️ Cannibals must **never outnumber** missionaries on either bank
- 🧠 Solved optimally in **11 moves** — can you match the AI?

## ✨ Features

- 🌊 **Animated 3D water** with custom shaders
- 🌲 **Living environment** — trees sway, fireflies glow, sun shines
- 🎭 **Stylized characters** — missionaries with halos, cannibals with feather headdresses
- 🤖 **BFS AI Solver** — get hints or watch the AI solve it step-by-step
- 🎉 **Victory confetti** & 💀 **game-over screen shake**
- ↩️ **Undo moves** anytime
- 📱 **Fully responsive**

## 🚀 Run Locally

```bash
npm install
npm run dev
```

| | |
